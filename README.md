# Algoritmos em C: Manipulação de Strings

Este repositório contém uma coleção de **20 algoritmos** para a manipulação de strings na linguagem C. O foco dos exercícios foi o domínio da **Tabela ASCII**, gestão de **memória (arrays)** e lógica de **ponteiros/índices**.

## Categorias de Desafios

### 1. Contagem e Estatísticas
* **Vogais e Consoantes:** Contagem seletiva ignorando caracteres especiais.
* **Dígitos e Maiúsculas:** Filtros baseados em intervalos ASCII.
* **Ocorrências:** Localização de caracteres específicos numa string.
* **Contador de Substrings:** Algoritmo para encontrar palavras dentro de frases (sem sobreposição).

### 2. Validações (Booleanos)
* **Palíndromo:** Verificação de leitura simétrica (ignora espaços e números).
* **Anagrama:** Comparação de duas strings usando técnica de marcação de caracteres.
* **Alfabético e Numérico:** Funções para validar se uma string contém apenas letras ou apenas números.

### 3. Transformações e Edição
* **Case Shifting:** Conversão total para Maiúsculas, Minúsculas e Capitalização de frases (Title Case).
* **Inversão:** Algoritmo de troca usando dois índices (início e fim).
* **Substituição:** Troca de caracteres antigos por novos em toda a string.
* **Iniciais:** Extração de letras maiúsculas de nomes completos.

### 4. Limpeza e Localização
* **Remover Espaços:** Limpeza inteligente de espaços nas extremidades (*Trim*) mantendo a integridade do meio.
* **Índices:** Funções para encontrar a primeira e a última posição de um caractere.
